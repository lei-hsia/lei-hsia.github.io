maven: java class missing dependencies

Author: LEI XIA Date: 二月 8, 2020  15:38

Solution:

when missing packages in Java class, hovering over the package name when it’s showing Add Maven Dependencies, click it. Do not manually write dependencies in pom.xml

e.g.

Using thrift file to automatically generate Java class, but many packages is missing. Click on “Add Maven Dependencies”

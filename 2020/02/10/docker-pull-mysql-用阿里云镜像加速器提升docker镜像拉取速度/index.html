docker pull mysql:用阿里云镜像加速器提升docker镜像拉取速度

Author: LEI XIA Date: 二月 10, 2020  14:54

docker pull mysql: pull 2小时还没拉取完…

注册阿里云“中国站”帐号，登录之后>”容器镜像服务”>镜像中心>镜像加速器.

每个人的加速器地址不同, 我的是: https://9mrtzyur.mirror.aliyuncs.com

Ubuntu 文档:

  1. 安装／升级Docker客户端

推荐安装1.10.0以上版本的Docker客户端

  1. 配置镜像加速器

针对Docker客户端版本大于 1.10.0 的用户

您可以通过修改daemon配置文件 /etc/docker/daemon.json 来使用加速器

配置完成之后再 docker pull mysql
